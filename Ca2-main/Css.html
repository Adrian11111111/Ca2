External style sheets

Devlopers tend to use seperate documents called a style sheet with css at the end of it. The .css lets the code understand that it is a stlye sheet and the link doesnt appear in the actual website it runs seperate to it but controlls what the user will see at the finished site, for example the developer can section out his work into diffrent classes and call upon them in the css file allowing him to set specific font styles, font sizes and loction on the screen and colors that show on the page like the background.
