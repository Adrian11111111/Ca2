Hyper Text Markup Language

What is HTML?

HTML stands for HyperText Markup language, Hypertext meaning a meta text that is used to refer to other pieces of text, and markup language meaning that it provides a series of elements and tags that instruct what the web browser should be doing with the text.

HTML is the standard system for tagging text files to create the structure for just about every webpage we use today, this includes links, tags, web page layouts and paragraphs. Each time we click on a web page on the internet, HTML will most likely be present.

Some common things HTML defines in websites include:

Paragraphs: One of the most common elements found in HTML and it defines a paragraph.

Headings: Each heading tag beginning with the letter h and ending in a number between 1 to 6, h1 being the biggest header and h6 being the smallest.

Block Elements: Allows spacing below themselves on a web page, display vertically on the left hand side of the webpage until styled by

HTML Features

Simple & User Friendly: Although HTML 5 comes with over 100s of built in tags into it only a handful of these tags actually get used regularly in normal web development

Presentation Friendly: It is easy to make an effective presentation with HTML due to amount of format tags there are to use.

HTML is platform independant: HTML runs off a browser and any device that you can find that has an operating system can run HTML
